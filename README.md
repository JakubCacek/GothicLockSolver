## Gothic Lock Solver

Gothic Lock Solver is a small web app for finding solutions to the lockpicking puzzles in Gothic Remake.  
The app takes the current lock positions and their dependencies, then searches for a valid sequence of moves that opens the lock.

Please use it only as a last resort, in a final act of desperation and frustration, when you would otherwise start searching for a ready-made solution on forums anyway. The goal is to help stuck players, not to encourage skipping every lockpicking challenge in the game.

https://gothiclocksolver.streamlit.app/

## A note on intended use

This app is meant as a last-resort helper for moments of desperation, not as a way to exploit the game.

Lockpicking puzzles are part of the intended challenge and atmosphere of the game. Using this solver for every chest can easily ruin the run, remove the sense of discovery, and flatten the experience. The goal of this project is not to encourage that.

The app was made mainly for players who would otherwise search for a ready-made solution online anyway. In that sense, it is meant to reduce frustration rather than replace the intended experience.

Use it deliberately, preferably only when you are truly stuck.

On the other hand, theft is an immoral act in itself. One might wonder whether cheating while committing an already immoral act makes the whole affair even more morally questionable, or whether, at that point, the ethical damage has already been done. Therefore, if you want to be a good person, you should not steal at all. You should join the Brotherhood of the Sleeper instead, and bring hope to the convicts of the Colony.
